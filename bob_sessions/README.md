# Bob Sessions

Hackathon judges require exported IBM Bob IDE task session reports in this folder.

Before upload:

1. Export the Bob IDE task session report.
2. Save it here.
3. Open the exported report and check it contains no API keys, cloud credentials, private data, or personal information.

Suggested name:

```text

```
